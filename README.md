## 使用系統

### 加入鴨發Go!

[https://line.me/R/ti/p/%40wap6503p](https://line.me/R/ti/p/%40wap6503p)

### 初始化環境

1. 設定`.env`檔
2. `docker-compose build`
3. `docker-compose run app bash ../script/init.sh`

### 運行

`docker-compose up`
